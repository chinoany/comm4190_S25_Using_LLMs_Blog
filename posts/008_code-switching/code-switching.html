Code-Switching: Testing ChatGPTs Ability to Switch Between AAVE, Cockney, and Gen-Z Lingo

Introduction

A single English sentence can sound like three different languages depending on dialect. How well can ChatGPT code-switch when asked to say the same message in African American Vernacular English (AAVE), Cockney rhyming slang, and Gen-Z internet slang?

Experiment Setup

  • Model: GPT-4o
  • Experiment:
    1. Enter Neutral Prompt
    2. Ask ChatGPT to rewrite it successively in AAVE, Cockney slang, and Gen-Z style.
    3. Evaluate its authenticity, fluency, and/or potential stereotyping.

Case 1

First prompt: Rewrite “I’m tired, but I’ll finish this project tonight.” in AAVE, Cockney rhyming slang, and Gen-Z slang (three separate lines).
GPT-4o chat response 1:
• AAVE: “I’m mad tired, but I’ma knock this project out tonight.”
• Cockney: “I’m cream-crackered, but I’ll bash out this caper tonight, no worry.”
• Gen-Z: “Low-key exhausted, but I’m grinding ’til it drops tonight 🔥.”

Observation:
- AAVE version uses “I’ma” and “mad tired” without caricature.
- Cockney inserts classic “cream-crackered” (crackered = knackered) and “caper”.
- Gen-Z line uses “low-key,” “grinding,” and the flame emoji. Usually people say “lowkey” as one word without the dash however I would say the use of grinding is spot on which is impressive because its a relatively new term for working super hard.

Case 2

Second prompt: Now rewrite “This new phone is expensive, but totally worth it.” in the same three dialects.
GPT-4o chat response 2:
• AAVE: “That new phone cost a grip, but it’s worth every penny.”
• Cockney: “That new dog-and-bone’s a bit dear, but proper worth the dosh.”
• Gen-Z: “Price is high, but the vibe? Priceless 💯.”

Observation:
- The AAVE version sounds a bit aged. Accurate but maybe in the 90s. - “Dog-and-bone” for phone fits Cockney rhyming slang tradition.
- The Gen-Z version doesn’t make much sense at all. Vibe is a Gen-Z term but it’s not necessarily used correctly here. Some might call this sentence a bit “cringe”.

Key Takeaways!

  1. Cockney rhymes and AAVE vocabulary were mostly accurate, but occasional missuses happened.
  2. Prompt clarity makes a difference. Chat gives specific responses when you explicitly labeling the dialect output you want
  3. Ethical vigilance needed. Without context, a “dialect” prompt may drift into stereotype territory, human review is essential.

Why Does This Matter?

For inclusive marketing or other things, you may need the same message tuned to diverse linguistic communities. ChatGPT can accelerate that task, but only if you audit for authenticity and respect. Dialect isn’t costume; it should be used with care and context.
